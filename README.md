# Ticketting
한번 티켓팅 할수 있는 프로그램을 만들어보자. 안되면, 임의 티케팅 사이트 만들어서라도 해보자

# How to start
```bash
git clone https://github.com/TakSung/Ticketting.git
cd Ticketting
pip install -r requirements.txt
```
